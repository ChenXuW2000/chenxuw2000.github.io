在Windows系统下的hugo配置/个人博客搭建

Feb 20, 2021
1 min read

Windows 下配置 hugo 建立个人blog 基于go语言

1.安装

从google github 下载zip file，解压在eg C：\Hugo\bin 添加hugo.exe的环境变量

set%PATH%=PATH; C：//Hugo/bin/hugo.exe

使用windows+R+cmd 打开命令行，

hugo help

2.建立blog folder

自定义名称

hugo new site myblog

3.下载theme

可google hugo themes 下载自己喜欢的主题； 根据网页指导进行theme安装; eg.

git clone xxxxx

Notes:下载theme前一定要配置好git git init

4.启动blog

在根目录中进行, eg. C：\myblog 本地启动命令:

hugo server -t m10c --buildDrafts

Ctrl+c 停止命令

复制网址到浏览器内，跳转显示blog；

修改blog内容，使用vscode打开eg. C:\myblog\content\post\blog.md ， 可修改title 和内容，修改完成后，再次使用启动命令，刷新网页即可。

5.将blog部署到github远端仓库上

第一步,创建一个新的repository，名字一定要是你的昵称的小写全拼， eg. ChenXuW2000—>chenxuw2000.github.io

第二步，cmd操作，在根目录eg. C：\myblog 进行， 使用命令: eg.

hugo --theme=m10c --baseUrl="https://chenxuw2000.github.io/"  --buildDrafts

此时会生成一个叫public的folder，切换到此folder，

git init
git add .
git commit -m“first attempt”
git remote origin https：//xxxx （目标文件的路径）
git push/git push -u origin master
